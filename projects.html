LSTD

light-std

A C++ library that aims to replace the standard C++ library with a much much simpler and, in my opinion, understandable API.

It is completely stand-alone and written from scratch with simplicity in mind.

light-std-graphics

An extension to lstd. Provides high-level APIs for displaying and managing windows, creating and drawing graphics.

The low-level graphics API we currently use is DirectX.

Simulating Rigid Body Physics (C++ and Python)

This notebook is an in-depth tutorial on the mathematics behind implementing a rigid body physics engine. We'll explore how you can take equations from physics and apply them in discrete time steps, how to check for intersecting objects, and how to generate a collision response.

We will interop two progrmaming languages so we can use C++ for rendering and Python for the simulation. This lets us focus on the logic and not deal with the quirks and verbosity of low-level languages.

When simulating physics we show interaction between objects graphically and try to model real-life phenomena - mass, forces, acceleration and velocity, soft/rigid bodies, even fluids, heat flow etc. This is very useful and has enormous practical applications for engineering, physics, chemistry and of course for entertainment (e.g. CGI in movies and games).

Classifying Fake News in Bulgarian (Machine Learning)

In this project we explore ways to classify news written in Bulgarian in two classes: real or "fake" (click-bait), using standard supervised machine learning methods.

Two types of features are used: the text content, which is vectorized with TF-IDF, and engineered features - Flesch Readibility Ease, words to character ratio, average word length, average sentence length, etc.

The result is about 82% accuracy and "real" news seems to be the easier class. The model is serialized and ready to be deployed with a small Python program and Flask.

Exploring Movies and Visualization (Data Science)

Here we do exploratory data analysis on a large movie dataset. We tidy and cleanup the data and then check pre-formulated hypothesis on it.

Are more movies being released in recent years?
Are certain genres more popular throughout the years?
Does runtime depend on the genre of the movie?
Is there a link between runtime and quality of the movie?
Is there a link between movie genres and poster color intensity?

The short answer to all the questions above is yes (except the last one). However the long answer is not so simple. Many factors might come into play and to say there is direct correlation is rather naive.

We also explore geography (countries in which movies are released) and all of this is accompanied by pretty plots and diagrams.
